The JAX-RS JSON-B Sample Application

This sample demonstrates how to use the Java API for JSON Binding with JAX-RS.

Description

This sample shows a JAX-RS application that produces and consumes JSON using the Java API for JSON Binding (JSON-B).

Key Features

This sample demonstrates the following key features:

  • The JSON Binding API: Serialization and deserialization of object.

Building, Deploying, and Running the Application

Perform the following steps to build, deploy, and run the application:

  1. Set up your build environment and configure the application server with which the build system has to work by following the common build instructions.
  2. app_dir is the sample application base directory: samples_install_dir/jsonb/jaxrs.
  3. Build, deploy, and run the sample application using the run outcome.

    mvn clean verify cargo:run

  4. The resources of the sample application are located at the following URLs: (The port number might vary.)

    • Jsonb serialization demo: http://localhost:8080/jsonb-jaxrs-resources/serialization
      Demonstrates JSON-B serialization

      • Using of default mapping
      • Using customizations
      • Using formatting
      • Using JsonbProperty and JsonbNillable annotations
    • Jsonb deserialization demo: http://localhost:8080/jsonb-jaxrs-resources/deserialization
      Demonstrates JSON-B deserialization and how JSON-P and JSON-B can be used together.

      • Demonstrate how JSON-P and JSON-B can be used together
      • Read Json document using JSON-P
      • Use JsonPointer to get an object
      • Use JSON-B to deserialize this object to POJO
    • Jsonb generic deserialization demo: http://localhost:8080/jsonb-jaxrs-resources/generic
      Demonstrates deserialization of generic list.

      • Deserialize jasons.json into List<Person>
  5. Use the clean outcome to undeploy the sample application and to remove the temporary directories such as build and dist.

    mvn clean
